The artifact I have chosen is my final project for CS-340, an animal shelter analytical dashboard written in Python with data stored in a MongoDB database. This project, created in April 2025, demonstrates my ability to build data-driven applications and integrate non-relational databases into software solutions.

To improve the project, I added a SQL database that allows the animal shelter to track volunteers and manage their shifts, showcasing my versatility with both relational and non-relational databases.

Code Enhancement

    
-- Volunteers Table
CREATE TABLE volunteers (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL
);

-- Shifts Table
CREATE TABLE shifts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    date TEXT NOT NULL,          -- ISO format 'YYYY-MM-DD'
    start_time TEXT NOT NULL,    -- format 'HH:MM'
    end_time TEXT NOT NULL       -- format 'HH:MM'
);

-- Assignments Table
CREATE TABLE assignments (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    volunteer_id INTEGER NOT NULL,
    shift_id INTEGER NOT NULL,
    FOREIGN KEY (volunteer_id) REFERENCES volunteers(id),
    FOREIGN KEY (shift_id) REFERENCES shifts(id)
);
